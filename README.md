# Multan – The City of Saints

A simple cultural webpage dedicated to **Multan**, one of the oldest cities in South Asia and famously known as *The City of Saints*.
The page highlights some key aspects of Multan’s identity including its **historic Sufi shrines, famous mangoes, and traditional crafts**.

This project was created as part of my **frontend development learning journey**.

## Features

* Hero section introducing Multan
* Cultural highlights of the city
* Interactive hover effects and transitions
* Circular cultural image cards
* Creator profile card
* Clean minimalist layout

## Built With

* **HTML5**
* **CSS3**
* Flexbox
* CSS hover interactions and transitions

## Purpose of This Project

This project was created to practice:

* Structuring a webpage with semantic HTML
* Layout design using Flexbox
* Creating reusable card components
* Adding interactive hover animations
* Building visually clean UI sections

It is part of my **web development practice while learning through Scrimba and building portfolio projects.**
